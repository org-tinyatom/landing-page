# TinyAtom Landing Page

Static marketing page for TinyAtom. No build step required.

## Preview

```bash
cd Tinyatom-landing-page
python3 -m http.server 8080
```

Open http://localhost:8080

## Theme

Uses the TinyAtom VS Code-inspired palette. Defaults to **System** theme via `prefers-color-scheme`.

## Deploy

Upload the folder to any static host. No dependencies.
